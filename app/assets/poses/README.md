# ポーズ画像アセット

このディレクトリには、プリセットポーズの画像を配置します。

## 必要な画像

以下のポーズ画像を用意してください：

| ファイル名 | ポーズ | 説明 |
|-----------|--------|------|
| `front.png` | 正面 | 正面を向いて立っている |
| `side.png` | 側面 | 横向きに立っている |
| `walking.png` | 歩行 | 歩いているポーズ |
| `sitting.png` | 座位 | 座っているポーズ |
| `arms_crossed.png` | 腕組み | 腕を組んでいる |
| `hands_on_hips.png` | 腰に手 | 腰に手を当てている |
| `casual.png` | カジュアル | リラックスした姿勢 |
| `professional.png` | プロフェッショナル | フォーマルな姿勢 |

## 画像仕様

- **形式**: PNG（透過背景推奨）
- **サイズ**: 200x300px 程度
- **内容**: 人物のシルエットまたはスケルトン図

## 一時的な代替案

画像が用意できない場合、システムは以下のフォールバック動作をします：
1. テキストラベルのみを表示
2. デフォルトのアイコンを表示
3. 既存のテキスト選択に戻す
