# 背景画像アセット

このディレクトリには、プリセット背景の画像を配置します。

## 必要な画像

以下の背景画像を用意してください：

| ファイル名 | 背景 | 説明 |
|-----------|------|------|
| `white.png` | 白 | 純白の背景 |
| `studio.png` | スタジオ | 写真スタジオ風の背景 |
| `gray.png` | グレー | グレーの背景 |
| `city.png` | 街 | 都市の背景 |
| `nature.png` | 自然 | 自然/森の背景 |
| `beach.png` | ビーチ | 海辺の背景 |
| `indoor.png` | 室内 | 室内の背景 |
| `abstract.png` | 抽象 | 抽象的な背景 |

## 画像仕様

- **形式**: JPEG または PNG
- **サイズ**: 300x200px 程度（サムネイル）
- **内容**: 実際の背景またはイメージ画像

## 一時的な代替案

画像が用意できない場合、システムは以下のフォールバック動作をします：
1. 色付きの矩形を表示
2. テキストラベルのみを表示
3. 既存のテキスト選択に戻す
