  Virtual Fashion Try-On v2.0 - 使い方ガイド
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【このソフトでできること】

服の画像をアップロードすると、その服を着たファッションモデルの
画像を自動生成します。

✨ v2.0の新機能！
・ポーズを画像で選択できる
・背景を画像で選択できる
・異なる角度から画像を生成できる
・チャットで画像を修正できる


【使い方（3ステップ）】

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
ステップ1: アプリを起動
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

VirtualFashionTryOn.exe をダブルクリック

※起動に20～60秒かかる場合がありますが、正常です。
※初回起動時のみ、APIキー設定ダイアログが表示されます。
　「はい」をクリックすると自動で設定されます。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
ステップ2: 服の画像を追加
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 「+ 画像を追加」ボタンをクリック

2. 服の画像ファイルを選択（PNG、JPG対応）

3. 衣類タイプを選択
   - トップス（Tシャツ、シャツ等）
   - ボトムス（パンツ、スカート等）
   - アウター（ジャケット等）
   - ワンピース
   - アクセサリー


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
ステップ3: モデル属性を選択
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✨ タブで切り替えて設定します

【基本タブ】
   - 性別: 女性 / 男性
   - 年代: 20代、30代等
   - 地域: アジア、ヨーロッパ等
   - 体型: 標準、スリム等

【ポーズタブ】✨ 新機能！
   - 画像をクリックしてポーズを選択
   - 8種類のプリセットポーズ
     (正面、側面、歩行、座位、腕組み、腰に手、カジュアル、フォーマル)
   - または「カスタムポーズ画像をアップロード」
     → 人物画像からポーズを自動認識！

【背景タブ】✨ 新機能！
   - 画像をクリックして背景を選択
   - 8種類のプリセット背景
     (白、グレー、スタジオ、街、自然、ビーチ、室内、抽象)
   - または「カスタム背景画像をアップロード」


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
ステップ4: 生成設定
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. サイズを選択（1024x1024 等）

2. 枚数を選択（1～4枚）

3. 生成モードを選択 ✨ 新機能！
   
   (●) 種類違い（バリエーション）
       → 同じ条件で異なるバリエーションを生成
   
   ( ) 角度違い（マルチアングル）
       → 同じモデル・衣類で異なる角度から撮影
       
       【角度違いモードの枚数別角度】
       - 1枚: 正面のみ
       - 2枚: 正面 + 背面
       - 3枚: 正面 + 側面 + 背面
       - 4枚: 正面 + 斜め前 + 側面 + 斜め後ろ

4. 「生成開始」ボタンをクリック


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
ステップ5: 結果の確認
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 待つ（10～30秒程度）

2. 完成！結果が表示されます

3. 保存するか、チャットで修正するか選択


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✨ 新機能: チャットで画像を修正
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

生成された画像を、自然な言葉で修正できます！

【使い方】

1. ギャラリーで修正したい画像の下にある
   「この画像を修正」ボタンをクリック

2. 右側のチャットパネルに画像が表示される

3. 修正内容をテキストで入力
   
   例:
   「もっと明るくして」
   「背景を青空に変更」
   「笑顔にして」
   「もっとカジュアルな雰囲気で」

4. 「送信」ボタンをクリック

5. AIが指示を理解して、修正画像を生成

6. チャットに結果が表示される

7. さらに修正したい場合は、また指示を入力
   （何度でも修正できます！）


【チャット修正の例】

ユーザー: 「もっと明るくして」
AI: 「承知しました。明るさを調整して再生成します」
[修正画像が表示される]

ユーザー: 「背景を森にして」
AI: 「背景を自然の森に変更します」
[さらに修正された画像が表示される]

ユーザー: 「完璧！」
AI: 「ありがとうございます！」


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
結果の保存
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 生成結果エリアの「保存」ボタンをクリック

2. 保存先フォルダを選択

3. すべての画像がPNG形式で保存されます
   （チャットで修正した画像も含む）


【ヒント】

・服の画像は背景が白や単色のものが最適です

・複数の服を追加すると、コーディネートした姿が生成されます

・角度違いモードを使うと、衣類を様々な角度から確認できます

・チャット修正で、細かい調整が簡単にできます

・ポーズや背景は画像で選択できるので、イメージしやすい！


【v2.0の新機能まとめ】

✨ ポーズ画像ギャラリー
   → 8種類のポーズを画像で選択
   → カスタムポーズ画像のアップロード（自動認識）

✨ 背景画像ギャラリー
   → 8種類の背景を画像で選択
   → カスタム背景画像のアップロード

✨ 角度違い生成モード
   → 同じ衣類・モデルで複数角度から撮影
   → 1～4枚で最適な角度を自動選択

✨ チャット修正機能
   → 「もっと明るく」等の指示で修正
   → AIが自動的に理解して再生成
   → 何度でも修正可能


【注意事項】

・インターネット接続が必要です

・画像生成には10～30秒程度かかります

・チャット修正も10～30秒程度かかります

・角度違いモードでは、枚数×生成時間がかかります
  （4枚の場合、約40～120秒）


【よくある質問】

Q: ポーズ画像が表示されません
A: 画像アセットが未配置の場合、絵文字とテキストで代替表示されます。
   機能には問題ありません。

Q: チャット修正が効きません
A: より具体的な指示を出してみてください。
   例: 「明るくして」→「照明を明るくして、全体的に明るい雰囲気で」

Q: 角度違いモードで同じ画像が生成される
A: AIの理解度に依存します。複数回試すか、ポーズを変更してみてください。

Q: チャットで何を指示すればいいですか？
A: 自然な言葉で大丈夫です！
   「明るく」「暗く」「笑顔」「真剣」「青空」「森」など。
